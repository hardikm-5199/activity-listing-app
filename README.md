# Activity Listing App

Cross-platform Activity Listing screen for an online learning platform.

## Tech Stack

- Expo (React Native)
- React Native Paper
- TypeScript
- React Native Web

## Features (In Progress)

- Activity listing (Classes & Assessments)
- Web + Native from same codebase
- Typed data model
